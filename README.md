Dojo-TDD
========

Coding dojo about TDD

Prerequisite
------------
* An IDE, Eclipse or Intellij
* JDK > 1.6 Installed

Eclipse
-------

   1. First you must generate eclipse project using gradle
    
    (windows) gradlew.bat eclipse
    (osx/unix) ./gradlew eclipse
 
    2. Then Open eclipse -> Import project
